# Part 2 - Communicating with Server

This part focuses on fetching data, updating server state, building reusable components, and handling forms and notifications in React applications.

## Projects

| Path | Focus |
| --- | --- |
| `courseinfo` | component extraction and data shaping |
| `phonebook` | CRUD UI, filtering, forms, and notifications |
| `country` | external API integration and weather lookup |

## Covered Work

- fetching data from APIs
- syncing UI state with server state
- building controlled forms
- extracting reusable components and hooks
- handling success and error notifications

## Run Locally

Each project is a separate Vite app:

```bash
cd part2/<project>
npm install
npm run dev
```
